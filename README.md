# Gaode-amap-api-poi
## 基于高德api-key提取相应分类的信息，采用行政区划编码提取，会先提取该行政区划下所有分类码的数据，再提取下一行政区，建议采用多个key轮流，测试一个key能提取8000-10000条左右数据。
## 带图形界面，方便处理。

##　高德个人开发者key申请、POI分类表、城市编码表地址：https://lbs.amap.com/api/webservice/download
